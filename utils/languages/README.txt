# VIP Transfer Bot – Locale Pack (TR, EN, RU, DE, AR)

Bu paket, 5 dil için **aynı anahtar yapısına** sahip JSON dosyaları içerir.
Klasör yapısı:
```
/locales
  ├─ tr.json
  ├─ en.json
  ├─ ru.json
  ├─ de.json
  └─ ar.json
```

## Notlar
- Arapça RTL olduğu için mümkünse Telegram UI'da yönü `rtl` olarak işleyin.
- Yer tutucular: `{name}`, `{phone}`, `{date}`, `{time}`, `{from}`, `{to}`, `{pax}`, `{baby_seat}`, `{extras}`, `{flight}`.
- Tüm metinler buton tabanlı akışa ve geri butonlarına uygundur.
